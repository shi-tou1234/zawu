# 🛠️ Zawu (杂物间) - 趣味小工具与项目合集

这是一个个人练习和趣味项目的集合仓库，包含网页特效、实用小工具以及 C++ 小游戏。

## 🌟 项目亮点

### 🌐 网页工具 (HTML/JS)
- **抽签工具 (`chouqian.html`)**：一个简单的在线抽签/决定助手。
- **粒子倒计时 (`particle_countdown.html`)**：炫酷的粒子动画倒计时效果。
- **圣诞树 (`圣诞树初级/高级.html`)**：不同等级的动态圣诞树网页展示。
- **英语辅助 (`英语朗读/翻译.html`)**：基于 Web API 的简易朗读与翻译工具。

### 🎮 本地程序 (C++)
- **五子棋 (`五子棋运行.cpp`)**：一个可以在终端运行的经典五子棋小游戏。

## 🚀 如何使用
1. **网页工具**：直接下载 `.html` 文件，使用任何现代浏览器打开即可。
2. **C++ 程序**：使用支持 C++11 及以上标准的编译器（如 Dev-C++ 或 VS Code）编译运行即可。
